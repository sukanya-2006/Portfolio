# 🚀 Sukanya Portfolio

A modern, interactive portfolio website showcasing my journey as a full-stack developer exploring AI-powered applications.

Built with a focus on clean UI, smooth animations, and real-world projects.

---

## ✨ Features

* ⚡ Smooth animations using Framer Motion
* 🎨 Clean, modern UI with custom styling
* 📱 Fully responsive design
* 📬 Functional contact form (EmailJS integration)
* 🧠 Showcases full-stack & AI-powered projects
* 🧭 Multi-page navigation with Next.js App Router

---

## 🛠️ Tech Stack

* **Frontend:** Next.js, React
* **Styling:** CSS / Tailwind (if used)
* **Animations:** Framer Motion
* **Email Service:** EmailJS
* **Deployment:** Vercel
